Introduction

The Math Pad is a mathematical formula editor you can use for notes or as a simple scratchpad. It consists of several mathboxes, where you can type in equations or text. The simple but powerful interface allows you to manipulate complex equations, all from the comfort of your own keyboard.

Commands and Symbols

The Math Pad nicely renders algebraic operations and mathematical symbols. Use / for fractions, ^ for exponents or superscripts, and _ for subscripts. Many LaTeX commands and symbols are also supported, such as \sqrt for square roots, \binom for combinatorics, \sum for summation, Greek letters such as \gamma and \Gamma (capitalization does matter here) and many more. For a more thorough reference, check out Art of Problem Solving's excellent documentation on LaTeX commands and symbols.

Keyboard Shortcuts

Enter Add new mathbox below
Backspace Delete and combine current and previous mathbox (if cursor at beginning of line)
Delete Delete and combine current and next mathbox (if cursor at end of line)
Arrow Keys Navigate within and between mathboxes
Control + Shift + Backspace Delete mathbox
Control + Alt + S Toggle save dialog
Control + Alt + H Toggle this dialog
